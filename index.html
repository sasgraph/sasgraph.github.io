Steph's HTML + CSS Cheat Sheet

HTML Interactive Tags

Tag Name Description
<details> Details Creates a disclosure widget that users can open/close.
<Summary> summary Defines the summary heading for a <details>block.
<dialog> Dialog Native modal dialog box.
<progress> Progress Bar Progress bar for showing task completion.
<picture> Picture Lets you define multiple image sources for responsive images.
<object> Object Embeds external interactive content like Flash, PDF, etc.
